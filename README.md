# internscan
To detect fake internship or not

Project Name: InternScan

Description:
InternScan is an AI-powered internship verification and resume analysis platform. It helps students identify genuine internship opportunities by analyzing company details, stipend information, and internship descriptions. The platform also provides an AI Genuine Score to help users avoid fake or suspicious internships.

Problem:
Many students struggle to find trustworthy internships online and often face fake job postings or unclear internship information.

Solution:
The platform automatically:
Analyzes internship details
Generates an AI Genuine Score
Detects suspicious internship postings
Helps students verify company information
Provides resume analysis support

Technologies Used:
React.js
Node.js
Express.js
MongoDB
Python
AI-based analysis modules

How to Run:
Install Node.js and MongoDB
Clone the project repository
Install dependencies using: npm install
Start the frontend and backend servers: npm start

Example:
Before:
Students manually verify internships from different websites.

After:
InternScan instantly provides:
AI Genuine Score
Internship verification
Resume analysis support
